# I.    PROJECT TITLE
------------------------------------------
## GABIFY


# II.   PROJECT DESCRIPTION
------------------------------------------
Gabify is a chat application to help seniors socialize and stay mentally active with content for users to promote nostalgia and reminisce about their past.

------------------------------------------
